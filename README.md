🐍 Logística de Programação: Minha Jornada com Python
Este repositório é o meu "laboratório" pessoal de estudos. Aqui, documento cada passo da minha evolução em lógica de programação utilizando Python, focando em construir uma base sólida para me tornar um Desenvolvedor Júnior.

🚀 O que estou construindo?
Meu objetivo aqui não é apenas escrever código que funcione, mas entender como o computador pensa. Estou saindo da sintaxe básica para estruturas de controle mais complexas.

🧠 Conceitos que estou dominando:
Entrada e Saída de Dados: Manipulação de informações com input() e print().

Estruturas de Decisão: Uso estratégico de if, elif e else para criar fluxos de escolha (como na minha Calculadora e no Radar de Velocidade).

Laços de Repetição: Controle de fluxo com while, garantindo que programas rodem de forma contínua e eficiente.

Tratamento de Dados: Conversão de tipos (int, float) e operadores aritméticos (incluindo o operador de resto %).

Prevenção de Erros: Iniciando estudos em try/except para criar códigos mais robustos.

🛠️ Projetos em Destaque
Calculadora Interativa: Um script que realiza operações matemáticas e só encerra quando o usuário decide, utilizando loops e condicionais.

Radar de Velocidade: Sistema de verificação de limites de velocidade com faixas de punição e alertas ao motorista.
